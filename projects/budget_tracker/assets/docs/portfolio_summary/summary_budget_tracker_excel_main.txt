This Excel file acts as both the primary front-end UI for data entry and the back-end data warehouse for all the financial transactions. In addition to that, it also contain 2 analytical tools; the trial balance and account ledgers. There are 4 core componenets; the journal (for transaction data), analytical tools (2 mentioned previously), 4 planners (asset, revenue, expenses, liability), and then reference tables.<br><br>

***DISCLAIMER*** All financial information shown was generated with the help of ChatGPT and does not reflect the personal information of any individual.<br><br>
